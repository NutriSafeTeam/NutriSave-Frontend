Conheça o projeto

NutriSafe, com o lema "comer com confiança é para todos", tem o intuito deNutriSafe, com o lema "comer com confiança é para todos", é um aplicativo que visa retornar ao usuário os ingredientes presentes em determinado produto, seja pelo scanner do código de barras ou pelo número digitado, e comparar com os ingredientes restritos informados pelo usuário

Objetivos


Facilitar o usuário na escolha de seus alimentos Exibir informações nutricionais Apresentar um resumo sobre o ingrediente predominante
